# Java Utils
## Simples classes that helps doing Java actions


### How to Install

#### Gradle

```kts
repositories {
   maven("https://jitpack.io")
}

dependencies {
    api("com.github.EduardMaster:java-utils:v1.0.0") // v1.0-version
}
```
#### Maven

```xml
<repositories>
    <repository>
        <id>Jitpack</id>
        <url>https://jitpack.com/</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>com.github.EduardMaster</groupId>
        <artifactId>java-utils</artifactId>
        <version>v1.0.0</version>
        <scope>provided</scope>
    </dependency>
</dependencies>
```
